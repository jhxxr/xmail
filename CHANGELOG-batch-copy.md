# 批量复制邮箱功能更新

## 功能说明

在管理员端邮箱列表页面新增了 **批量复制** 功能，支持将多个邮箱的账号密码以自定义分隔符格式批量复制到剪贴板。

## 使用步骤

1. **选择邮箱**
   - 在邮箱列表页面勾选需要导出的邮箱
   - 可以使用全选功能批量选择

2. **点击"复制选中"按钮**
   - 选中邮箱后，顶部会显示 "复制选中 (N)" 按钮
   - 点击该按钮打开批量复制配置弹窗

3. **配置分隔符**
   - **预设分隔符**：提供 5 种常用分隔符
     - 冒号 (`:`)
     - 横线 (`----`)
     - 逗号 (`,`)
     - 竖线 (`|`)
     - Tab (`	`)
   
   - **自定义分隔符**：可以输入任意自定义分隔符
     - 例如：`@@`、`###`、` -> ` 等

4. **预览格式**
   - 配置区域会实时显示格式预览
   - 格式：`邮箱地址{分隔符}密码`

5. **确认复制**
   - 点击 "复制到剪贴板" 按钮
   - 系统会自动获取所有选中邮箱的密码
   - 一行一个，格式如下：

```
test01@example.com:password123
test02@example.com:password456
test03@example.com:password789
```

## 功能特点

- ✅ **灵活的分隔符选择**：支持 5 种预设 + 自定义
- ✅ **一行一个**：每个邮箱账号密码独立一行
- ✅ **批量处理**：自动并行获取所有邮箱密码
- ✅ **实时预览**：选择分隔符后立即显示格式预览
- ✅ **错误处理**：如果某些邮箱密码获取失败，会单独提示
- ✅ **剪贴板复制**：一键复制所有内容到剪贴板

## 使用场景

1. **批量导入其他系统**
   - 使用冒号分隔符 (`:`) 适合很多认证系统
   - 使用逗号分隔符 (`,`) 适合 CSV 导入

2. **脚本自动化**
   - 使用 Tab 分隔符适合脚本解析
   - 使用自定义分隔符匹配特定工具格式

3. **文档记录**
   - 使用横线或其他可读性强的分隔符
   - 便于人工阅读和整理

## 技术实现

- **并行获取密码**：使用 `Promise.all` 并行处理多个邮箱，提高效率
- **安全处理**：通过后端 API 获取密码，不直接暴露在前端
- **用户友好**：提供加载状态、错误提示、成功反馈等完整交互

## 与导出功能的区别

| 功能 | 批量复制 | 导出 CSV |
|------|---------|---------|
| 输出格式 | 纯文本，自定义分隔符 | CSV 文件 |
| 分隔符 | 可自定义（5种预设+自定义） | 固定逗号 |
| 使用场景 | 快速粘贴到其他工具 | 文件归档、表格处理 |
| 操作步骤 | 选择→配置→复制 | 选择→导出 |
